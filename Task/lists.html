Розшифровка тегів

  1. Список визначень:
    2. dl
    Definition List, сам список определений
    dt
    Definition Term, термин
    dd
    Definition Definition, определение термина

  2. Нумеровані списки:
    3. ol
    Ordered list - упорядоченный список

  3. Макровані списки:
    4. ul
    Unordered List, неупорядоченный список
    li
    List Item, элемент списка

АТРИБУТИ:
  1. reversed:
    2.         Указывает, что элементы списка будут идти в порядке убывания (вместо порядка по возрастанию).
            Возможные значения логического атрибута:
        1
        2
        3
        Примечание: атрибут reversed поддерживается только браузерами Chrome и Safari.
  2. start:
    3.         Устанавливает начальное целочисленное значение, с которого будет начинаться нумерация элементов в списке.
  3. type:
    4.         Определяет вид маркера, который будет использоваться для элементов списка:
        1 - десятичные числа (1, 2, 3, 4 ...).
        A - латинские буквы в алфавитном порядке, в верхнем регистре (A, B, C, D ...).
        a - латинские буквы в алфавитном порядке, в нижнем регистре (a, b, c, d ...).
        I - римские цифры в верхнем регистре (I, II, III, IV ...).
        i - римские цифры в нижнем регистре (i, ii, iii, iv ...).
